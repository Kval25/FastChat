A real-time chat application built using **Swift and Firebase**, designed for fast and seamless messaging with a clean user interface.

---

## 🚀 Features

- 💬 Real-time messaging using Firebase Firestore  
- 🔐 User Authentication (Register & Login)  
- ⚡ Instant message updates  
- 🎨 Clean and responsive UI  
- ☁️ Cloud-based data storage  

---

## 🛠️ Tech Stack

- **Language:** Swift  
- **Framework:** UIKit  
- **Backend:** Firebase  
  - Firebase Authentication  
  - Cloud Firestore  
- **IDE:** Xcode
